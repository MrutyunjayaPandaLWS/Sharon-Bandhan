• மொத்த தொகை ரூ. 20,000 (1 புள்ளி = 1 ரூபாய்) மதிப்புள்ள புள்ளிகளைத் தாண்டியவுடன், மூலத்தில் வரி விலக்கு (TDS) மீட்பு மதிப்பின் மீது பொருந்தும்:

    - பான் எண் இருந்தால் 10%

    - பான் எண் இல்லை என்றால் 20%


  • வெகுமதி புள்ளிகள் மூலம் பெற்ற பரிசை வாடிக்கையாளருக்கு வழங்கப்பட்டவுடன் TDS கழிக்கப்படும்.

  • கழிக்கப்பட்ட TDS திரும்பக் கொடுக்கப்படாது. கழிக்கப்பட்ட டிடிஎஸ் இந்திய அரசாங்கத்திற்கு செலுத்தப்படும்.
